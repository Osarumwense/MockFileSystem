User Stories:                                                                   Duration:

As a user I want to be able to execute a previously entered command.            | 1 day
As a user I want to be able to move a file from its current location to a
different file location, or directory.                                          | 2 days
As a user I want to be able to move a directory from its current location to a
different directory.                                                            | 3 days
As a user I want to be able to copy a directory with all the files inside to a
new directory									| 3 days
As a user I want to be able to copy a file from one directory to the other
directory									| 1 day
As a user I want to be able to read the files from any directory that I want    | 2 days
As a user I want to be able to print lines in files that contain a certain
REGEX that I provide along with a certain PATH					| 2 days
As a user I want to be able to print lines from a given path that contain a
REGEX that I provide along with the directory location printed followed by the
file name, then a colon, and the line itself.					| 2 days

As a user, I want to be able to add the contents of a file found at a given URL
web address to my working directory.                                            | 2 days
As a user, I want to be able to input a path and be given the entire contents
of that path, regardless of whether singular path sections represent files or
directories.                                                                    | 3 days
As a user, I want to be able to list the contents of my current working
directory.                                                                      | 1 day
As a user, I would like to be able to know all the subdirectories of my current
working directory.                                                              | 1 day
As a user, I want to have a file, which only contains the string type I enter   | 1 day
through standard input.
As a functional user of the program, I want to append the input string at the   | 1 day
end of my indicated file.
As a user, I want to read the documentation of a user requested feature and
the respective implementations and uses for a specific command line.            | 1 day
As a user, I want to be able to see all of the contents (files and directories)
relative to the current directory or when inputting a directory path            |3 days
As a user, I may also want to be able to check whether a directory exists or not
by calling a command to see if an error is raised                               | 3 days
As a user, I want to be able to create a single or several directories with
separate names relative to the current directory in one single input            | 3 days
As a user,I want to be able to view the contents of a single or several files
directly in the shell when given the location of the file(s)                    | 3 days
As a user, I want to be able to obtain the name of the current directory I am
operating within.                                                               | 1 day
As a user, I want to know all the parent directories of my current working
directory.                                                                      | 1 day
As a user, I want to be able to add a new directory to the repository and make
that new directory the one I am currently working in.                           | 3 days
As a user, if I add a new directory to the repository and switch into that as
my current working directory, I must have the ability to switch back into the
directory I was just in prior to the addition of a new directory.               | 3 days
As a user, I want to be able to be able to change directories freely. I want
to be able to move anywhere respective to my current directory or the path I
am currently located in.                                                        | 2 days
As a user I can close the program in order to end all operations when I am
finished.                                                                       | 1 day
As a user I can go back into my previous directory without having to type in the
path so as to make moving around directories easier and faster.                 | 2 days
As a user I can view all the previous commands I have entered in the order they
were entered, as well as the syntactical errors I've typed. This way I easily
know what commands I have entered, what I have done and in what order.          | 3 days
As a user I can also view a certain number of my last previous commands in the
order I entered them, so that I dont always have to view and sift through the
list each time, but I can look at a more specific list.                         | 2 days
